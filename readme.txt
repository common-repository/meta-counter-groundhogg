### ðŸ”„ How to use
* In the funnel view, when adding a new step to your funnel, a new action icon will appear called "Meta Counter". Select it to add it to your funnel to use it.
* Select the meta feild you would like to increase when the action runs. (make sure its a field that stores a number or create a custom feild with a default value of 0)
* Select the increment level you would like it to count by (eg: 1 will add 1 each time its ran, 2 will add 2 each times its ran)
* Use the "Feild Changed" benchmark and select the field you are updating to sense when the meta field changes and set targets to trigger new actions.
